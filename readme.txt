== Description ==

Display an animated countdown clock for the 2016 Rio de Janeiro Olympic Games

* Choice of colors for texts and background,
* Choice of clock layouts, e.g. vertical or horizontal,

== Installation ==

1. Download the zip file and extract the contents,
2. Upload the `olympics-countdown-clock` folder to your WP plugin folder `/wp-content/plugins/` directory
3. Go to Plugins > Plugins, and activate the plugin
4. Go to Appearance > Widgets, and drag the widget to a sidebar,
5. Open countdown clock's settings menu and customize the countdown,

== Frequently Asked Questions ==

= Can I paste the countdown onto my page on a post, rather than a sidebar =
Yes, the javascript code is available directly from  [mycountdown.org](http://mycountdown.org/Sports/2016_Olympics/getwidget/). You must paste the javascript into the post in HTML mode.

= Can I set the time of day and world timezone for the event ? =
Yes, select from the timezone dropdown select the time zone in your area.
